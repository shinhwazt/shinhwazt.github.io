改变this指向

在JS中我们可以使用三种方式来改变函数运行时的this的指向。分别为call、apply、bind

三种方法都可以改变this的指向，call、apply在使用时函数会立即执行，而bind则是创建一个新的 函数，并返回这个函数，函数内的this就是bind内传递的参数

call、apply在使用时也有不同。call可以接收多个参数，第一个参数表示this的指向，后面的参数 则表示函数的参数，apply最多只能接收两个参数，第一个参数表示this的指向，第二个参数可以是一个 普通变量或者是一个数组，表示函数的参数。

			function Bird(){
				this.word = "我是一只鸟";
			}
			Bird.prototype.fly = function(){
				var skill = "我会飞"
				alert(this.word+"->"+skill)
			}
			function Fish(){
				this.word = "我是一只鱼"
			}
			Fish.prototype.swim = function(){
				var skill = "我会游";
				alert(this.word+"->"+skill)
			}
			function Animal(){
				this.word = "我是？";
			}
			Animal.prototype.say = function(){
				alert(this.word);
			}

			var bird = new Bird();
			var fish = new Fish();
			Animal.prototype.say.call(bird);//我是一只鸟
			Animal.prototype.say.apply(bird);//我是一只鸟
			var method = Animal.prototype.say.bind(bird);method();//我是一只鸟

在上面的代码中call和apply表示执行Animal.prototype.say这个函数函数中的this为call和 apply的参数bird等价于bird.say()

bind表示复制一个Animal.prototype.say函数并把函数里面的this指向bird，并把这个函数返回去。
