6.导航

导航的使用方式为：

<ul class="nav nav-tabs">
  <li><a href="javascript:void(0)">导航一</a></li>
  <li><a href="javascript:void(0)">导航二</a></li>
  <li><a href="javascript:void(0)">导航二</a></li>
</ul>

效果如下：

  • 导航一
  • 导航二
  • 导航二

导航的使用方式为class="nav nav-tabs/tab-pills nav-stacked/nav-justified" class="nav"是导航的基本样式，class="nav-tabs/nav-pills"是导航的具体样式，class="nav-tabs"是上图效果中的导航、class="nav-pills"则是胶囊式导航， class="nav-stacked/nav-justified"则是对我们的导航做进一步的进化， class="nav-stacked"表示垂直导航，class="nav-justified"表示自适应的导航。

6.分页导航

BootStrap提供了两种分页导航，一种是带有多个页码的一种是只有上一页和下一页的。

6.1页码分页

页码方式的使用方式为：

<ul class="pagination">
  <li><a href="#">«</a></li>
  <li><a href="#">1</a></li>
  <li><a href="#">2</a></li>
  <li><a href="#">3</a></li>
  <li><a href="#">»</a></li>
</ul>

效果如下：

  • «
  • 1
  • 2
  • 3
  • »

6.2上一页，下一页

使用方式为：

<ul class="pager">
  <li><a href="#">上一页</a></li>
  <li><a href="#">下一页</a></li>
</ul>

效果如下

  • 上一页
  • 下一页
  • 上一页
  • 下一页

7.面包屑导航

面包屑导航在平时的开发中主要提示我们当前所处的页面的层级，同时提供给查看上面层级的超链接

使用方式如下：

<ul class="breadcrumb">
  <li><a href="javascript:void(0)">Home</a></li>
  <li><a href="javascript:void(0)">Library</a></li>
  <li class="active">Data</li>
</ul>

效果如下：

  • Home
  • Library
  • Data

8.导航条

导航条的使用方式为：

<div class="navbar navbar-default" role="navigation">
  <div class="navbar-header">
    <a class="navbar-brand" href="#">Brand</a>
  </div>
  <ul class="nav navbar-nav">
    <li class="active"><a href="#">active</a></li>
    <li><a href="javascrip:void(0)">Link</a></li>
    <li class="disabled"><a href="#">disabled</a></li>
    <li><a href="javascrip:void(0)">Link</a></li>
  </ul>
</div>

效果如下：

Brand
  • active
  • Link
  • disabled
  • Link

响应式导航条使用方式为：

<div class="navbar navbar-default" role="navigation">
  <div class="navbar-header">
    <a class="navbar-brand" href="#">Brand</a>
    <button type="button" class="navbar-toggle" data-toggle="collapse" data-target=".zt">
      <span class="icon-bar"></span>
      <span class="icon-bar"></span>
      <span class="icon-bar"></span>
    </button>
  </div>
  <div class="collapse navbar-collapse zt">
    <ul class="nav navbar-nav">
      <li class="active"><a href="#">active</a></li>
      <li><a href="javascrip:void(0)">Link</a></li>
      <li class="disabled"><a href="#">disabled</a></li>
      <li><a href="javascrip:void(0)">Link</a></li>
    </ul>
  </div>
</div>

效果为：(窗口缩小到超小分辨率)

Brand
  • active
  • Link
  • disabled
  • Link

相比于普通的导航发生了变化：在class="navbar-header"的容器里面添加了 <button>元素该元素的关键属性为class="navbar-toggle", data-toggle="collapse"，data-target=".zt"(data-target属性值是随意的但是要对应后面<div>的class或者id),后面的<ul>外面包裹了<div> 该元素属性为class="collapse navbar-collapse zt",前两个类名是固定的最后一个要和 data-target相对应。
